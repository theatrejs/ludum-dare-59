# Ludum Dare 59

> *🎮 A Theatre.js Game.*

## Installation

Clone the repository:

```shell
git clone https://github.com/theatrejs/ludum-dare-59.git
```

Intall the dependencies:

```shell
npm install
```

## Quick Start

```shell
npm start
```

## Edition

Create the `.env` file from the `.env.example` file:

```shell
cp .env.example .env
```

Update the `.env` file with your Aseprite and Reaper executable paths, for instance:

```
ASEPRITE=C:\Program Files\Aseprite\Aseprite.exe
REAPER=C:\Program Files\REAPER (x64)\reaper.exe
```
